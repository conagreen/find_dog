## 주제

반려견 입양에 앞서 다양한 성격과 성향을 지닌 강아지들 중 어떤 견종이 본인과 잘 맞을지 MBTI 검사를 통해 견종을 매칭해주는 서비스.

<br>

## 개발언어
- Backend: Java 8
- Frontend: React

<br>

## 개발환경
- Java: JDK 1.8
- IDE: IntelliJ
- DB : mySQL
- 빌드 관리 : Gradle
- 프레임워크: SpringBoot
> - ORM: Spring-Data-JPA
> - 보안 : Spring security
> - 형상 관리 툴 : git
> - Others: Websocket, Lombok

<br>

## 기능

1. mbti 검사 : https://github.com/DongHyunKIM-Hi/mbti
2. chat봇을 통한 상담
3. 게시글, 댓글 기본 CRUD 및 좋아요 기능
4. 회원가입 / 로그인

<br>
